API Reference
GET /export/json/{analysis_id}

Returns the complete analysis payload, including event counts, anomalies, and AI insights in JSON format. Suitable for integration with external SIEM tools.

GET /export/csv/{analysis_id}

Exports all identified critical events (Errors, Warnings, Failures, Timeouts) into a flat CSV format for spreadsheet analysis.

POST /ask

Query the AI intelligence layer using natural language. Requires JSON payload: {"question": "string", "analysis_id": "string"}.
